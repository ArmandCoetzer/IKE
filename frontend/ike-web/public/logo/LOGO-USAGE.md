# IKE logo assets

| Asset | File | Usage |
| ----- | ---- | ----- |
| Full / marketing | `ike-full.png` | Login, register, invite flows — above the form. |
| Icon / favicon | `ike-icon.png` | Navbar brand, favicon (`index.html`), compact spaces. |

Paths are under `public/`, served as `/logo/...` from the Angular app.

Replace `ike-full.png` with a wide lockup when you have one; the icon can stay circular.
