# Tradion logo usage

| Image | Filename | Where it's used |
|-------|----------|-----------------|
| Full logo (emblem + "Tradion" text) | `tradion-full.png` | **Header bar** — `main-layout.component.html` (navbar brand). Height 48px (fits 56px bar), max-width 200px; brand has padding so the emblem isn’t clipped. |
| Text-only ("Tradion" in brand color) | `tradion-text.png` | **Login** and **Register** — above "Sign in" / "Create an account". Height 104px, max-width 400px. Use for emails or other text-only contexts. |
| Icon only (geometric shape) | `tradion-icon.png` | **Favicon** — `index.html` `<link rel="icon">`. Use for browser tab, app icon, small spaces. |

All paths are relative to `public/`, e.g. `/logo/tradion-full.png`.
