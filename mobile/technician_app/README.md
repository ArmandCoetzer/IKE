# Ian Kleyn Electrical — IKE Technician App

Flutter technician client for the IKE field-service API.

## Branding

IKE palette (aligned with [ikegroup.co.za](https://ikegroup.co.za/)): brand red `#E31837`, near-black `#0A0A0A`, white.

## Run

From `mobile/technician_app`:

```bash
flutter pub get
flutter run
```

Configure API base URL in app settings / environment as documented for your deployment.
